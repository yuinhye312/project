- H2 Console Url: http://localhost:8080/h2

  - 사용자목록: /user/list

  - 계좌추가: /account/add
  - 계좌목록: /account/list

  - 계좌내역추가: /accountptcl/add
  - 계좌내역목록: /accountptcl/list
